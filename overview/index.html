Overview

Sneaker is a modern C++ library that contains a vast set of re-usable functions and objects that are useful in many application development scenarios.

Content

  • libc - utilities for C
  • allocator - STL container allocator
  • cache - in-memory cache management
  • container - high performance data structures
  • context - context management
  • functional - versatile function wrappers that support asynchronous invocations
  • threading - thread management and daemons

Get Started

Build binary

make all

Run all tests

make run

Clean all builds

make clean

Install

make install

Uninstall

make uninstall
